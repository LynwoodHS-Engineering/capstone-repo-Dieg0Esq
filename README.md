# EDD-Capstone


Our project is to be remote controlled and function effectively, integrating all 5 required elements into one system. Our main goal is to construct a properly-runnning tank with a fully functional turret, portraying a real life tank. 


Requirements: Project is to contain at least ine functional element from each of the 5 sections: Output Display, Manual User Input, Automatic Sensors, Mechanisms, Actuators & Hardware, and Logic, Processing, & Control. 


Our project successfully meets the requirements, consisting of a LED screen & LED's(Output Display), Remote Control(Manuel User Input), Accelerometer(Automatic Sensor), RC Servo Motors & 393 Motors(Actuators), and an Arduino Uno & ESP32(Logic, Processing & Control)
